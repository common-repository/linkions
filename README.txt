Linkions helps to add a social link page, so that you can share the page on social medias.

== Description ==

Linkions helps to create a seperate page on your website that helps to create a personal social linking page. You can add the name, descritpion, add image and can also set social media links via our plugin.

== Installation ==

* Visit WordPress Admin Dashboard > Plugins > Add New
* Search for 'Linkions'
* Click "Install now" and then "Activate" Linkions

Install manually:
1. Upload `linkions.zip` zip file to the `/wp-content/plugins/` directory
2. Go to WordPress admin > Plugins
2. Activate Linkions plugin from the Plugins list

== Frequently Asked Questions ==

= How to change the Social links within the Linkions? =

To change the social links you need to goto Linkions > Social tab

= What will happen if I set the same page URL for another page in WordPress? =

In this case, we will show a message to set a new URL within the Linkions page, you can set a new slug via our Linkions > Appearance page

= Is Linkions plugin GDPR Compliant? =

Yes, Linkions is fully GDPR Compliant. We does not use any cookies, tracking scripts, or store any user data.
